Trexland-VR project
2021
Technique : Unity, Blender
In this project, the user moves around using the mouse or VR controller. The user follows the instructions and follows the footprints of the Tyrannosaurus. When the user reaches the end of the path, he can find Tyrannosaurus and Tyrannosaurus eggs and nests. The user can interact by collecting the eggs scattered around the nest and gradually get different information about the Tyrannosaurus. (Speed, size, weight, etc.). Each time user collects the eggs in his nest, the dinosaur makes different movements, such as wagging his tail, howling and running.
